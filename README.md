🍽️ Site de Receitas
Site que consome a API do DummyJSON para exibir receitas de forma simples e interativa.
🔧 Tecnologias

HTML
Tailwind CSS
JavaScript

⚙️ Funcionalidades

Exibe receitas com imagem, ingredientes, instruções e avaliação
Botões para navegar entre as receitas (Voltar / Avançar)
Busca por nome ou ID da receita
Exibe imagem de "não encontrada" caso a busca não retorne resultado

🌐 API utilizada
DummyJSON Recipes
🚀 Como rodar

Clone ou baixe o projeto
Abra o arquivo index.html no navegador


Não precisa instalar nada.
